About this Website

This website is for SE/COM S 319 Spring 2023 at Iowa State University. It is for the class's midterm project, and the website contains a detailed list of some of our favorite video games. The game listings on the main "games" webpage are generated by dynamically reading the respective data from a JSON file. Additionally, each game's more detailed page is also generated from the data read in from the JSON file.
The project was developed by Andrew Sand and Nathan Geater.
